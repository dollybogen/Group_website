我们组期望完成可以将整个系统拆分为以下 6 大核心模块。我们组打算选择康复分析模块

主要功能和关键技术

1. 传感器数据采集模块（Data Collection Module）

功能：

  • 角度传感器（MPU-6050等）实时采集下肢运动数据
  • 通过 蓝牙低功耗（BLE） 传输数据至移动端
  • 传感器数据预处理（数据格式化、时间同步等）

关键技术：

  • 硬件接口（I2C/SPI 读取 MEMS 传感器数据）
  • BLE 无线通讯（蓝牙模块连接 & 低功耗数据传输）
  • 数据时间同步（传感器时间戳矫正）

2. 数据处理模块（Data Processing Module）

功能：

  • 过滤 & 去噪（减少传感器误差和环境干扰）
  • 计算 关节角度、运动轨迹、速度、加速度
  • 进行 数据插值、平滑处理（卡尔曼滤波）

关键技术：

  • 信号处理（卡尔曼滤波、滑动平均滤波）
  • 数学建模（欧拉角转换、运动学计算）
  • 优化算法（降低噪声，提升计算精度）

3. 数据管理 & 存储模块（Data Storage Module）

功能：

  • 将 患者康复数据 进行本地 & 云端存储
  • 维护 历史数据，支持趋势分析 & 预测
  • 提供 快速检索（按时间、康复进度查询）

关键技术：

  • 数据库存储（SQLite / Firebase / 云存储）
  • 数据索引（提高数据查询效率）
  • 数据同步（本地 & 云端数据一致性管理）

4. 康复分析 & 预测模块（Recovery Analysis & Prediction Module）

功能：

  • 计算 患者恢复进度（与正常康复曲线对比）
  • 进行 AI 预测（基于历史数据预测未来康复趋势）
  • 生成 个性化报告（康复数据可视化）

关键技术：

  • 机器学习（线性回归 / ARIMA 时间序列预测）
  • 统计分析（计算均值、标准差、恢复趋势）
  • 数据可视化（Matplotlib / D3.js 绘制图表）

5. 康复训练推荐模块（Rehabilitation Training Recommendation Module）

功能：

  • 根据患者 当前数据 生成 个性化康复训练方案
  • 提供 视频 / 动图 演示康复训练方法
  • 记录训练效果，优化 训练计划

关键技术：

  • AI 推荐算法（基于用户数据匹配最优训练方案）
  • 视频 / 动画播放（嵌入训练示范视频）
  • 用户反馈系统（患者训练后反馈 & 训练调整）

6. 用户交互 & UI 模块（User Interface & Mobile App Module）

功能：

  • 直观展示 患者实时运动数据
  • 显示 康复进度、训练计划、AI 预测
  • 提供 远程数据共享功能（医生查看患者数据）

关键技术：

  • 跨平台开发（Flutter / React Native / Android / iOS）
  • 数据可视化（ECharts / Chart.js）
  • 远程数据访问（RESTful API / WebSocket）
